# 🐝 Beeve - 국민체력100 기반 AI 개인 맞춤 헬스케어 플랫폼

“내 체력을 알면, 운동이 쉬워진다.” <br>
스마트폰만으로 6가지 체력 요소를 측정하고, 동년배 대비 순위/백분위와 AI 맞춤 운동 추천까지 제공하는 개인 맞춤형 헬스케어 플랫폼 <br>
국민체력100의 공인 데이터를 기준으로 객관적이고 비교 가능한 체력 관리 경험을 제공 <br>

## 🔎 프로젝트 개요

- **설명**: 국민체력100 데이터 기반 6각형 체력 측정(근력/근지구력 등) + 데이터 히스토리 + AI 맞춤 운동 추천 플랫폼

- **기간**: 2024.10.11 ~ 2024.12.07

- **구성**: BE 2명, FE/기획 1명, 디자이너 1명

## ✨ 핵심 기능

- 동영상 촬영 기반 6각형 체력 측정 기록: 근력/근지구력/심폐지구력/유연성/순발력/민첩성

- 체력 시각화 데이터 제공: 레이더 차트용 스케일/등급(1~5) + 동년배 비교(백분위)

- 측정 히스토리 관리: 날짜별 기록 조회 및 변화 추적

- AI 맞춤 운동 추천: 사용자 환경(장소/부상/질병) 반영한 루틴 생성 및 저장

## 🛠️ 나의 사용 기술 스택

### Backend

- Kotlin, Spring Boot, Spring Security, Spring Data JPA

- PostgreSQL, MongoDB

- Swagger, Postman, Git / GitHub


## 📌 Why Beeve

- 체성분(인바디) 중심의 정보는 실제 운동 능력(심폐/유연성/민첩 등) 판단에 한계

- 국민체력100은 신뢰도 높지만 센터 접근성과 시간 제약으로 지속 측정이 어려움 <br>
  → Beeve는 스마트폰 기반 측정/관리로 정기적인 체력 추적과 맞춤 운동 추천을 가능하게 함
